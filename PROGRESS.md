# SWaGi InTech...

## Project Overview

SWaGi InTech... is a multi-tenant Wi-Fi billing and ISP management platform built for the Kenyan market.

Core technologies:

- Cloudflare Workers
- Cloudflare D1
- Cloudflare Pages
- Cloudflare AI
- MikroTik RouterOS API
- Safaricom Daraja API
- TypeScript
